# AI-Shopping 后端代码规格说明

## 1. 技术栈

- **框架**: Spring Boot 3.2.3
- **Java版本**: Java 17
- **持久层**: MyBatis 3.0.3
- **数据库**: MySQL
- **缓存**: Redis
- **AI集成**: LangChain4j 0.35.0 (阿里云DashScope)
- **构建工具**: Maven

## 3. 代码规范

### 3.1 包命名规范

- 根包: `com.gzasc.aishopping`
- 子包按功能划分: `controller`, `service`, `mapper`, `model`, `dto`, `config`, `utils`
- 控制器分层: `controller/seller/`, `controller/user/`

### 3.2 类命名规范

| 类型 | 命名规则 | 示例 |
|------|----------|------|
| 实体类 | 名词，驼峰命名 | `Product`, `Order`, `Contact` |
| 控制器 | 名词+Controller | `ProductQueryController`, `OrderUserController` |
| 服务接口 | 名词+Service | `OrderService`, `ProductService` |
| 服务实现 | 名词+ServiceImpl | `OrderServiceImpl`, `ProductServiceImpl` |
| Mapper | 名词+Mapper | `OrderMapper`, `ProductMapper` |
| DTO | 描述性名词 | `PlaceOrderRequest` |

### 3.3 代码风格

#### 3.3.1 实体类 (Model)

```java
package com.gzasc.aishopping.model;

import lombok.Data;
import lombok.AllArgsConstructor;
import lombok.NoArgsConstructor;

@Data
@NoArgsConstructor
@AllArgsConstructor
public class Product {
    private String id;           // 商品ID
    private String name;         // 商品名称
    private Double price;        // 商品价格
    private String tags;         // 商品标签
    private String description;  // 商品描述
    private int stock;           // 商品库存
    private Date createdAt;      // 创建时间
    private Date updatedAt;      // 更新时间
}
```

**规范要点**:
- 使用 Lombok 注解 (`@Data`, `@NoArgsConstructor`, `@AllArgsConstructor`)
- 字段使用驼峰命名
- 字段后添加中文注释说明
- 优先使用包装类型 (`Double`, `Integer`) 而非基本类型

#### 3.3.2 控制器 (Controller)

```java
package com.gzasc.aishopping.controller;

import lombok.RequiredArgsConstructor;
import org.springframework.web.bind.annotation.*;
import java.util.Map;

@RestController
@RequestMapping("/api/product")
@RequiredArgsConstructor
public class ProductQueryController {
    
    private final ProductService productService;

    @GetMapping("/all")
    public Map<String, Object> getAllProducts(
            @RequestParam(name = "page", defaultValue = "0") int page) {
        try {
            List<Product> products = productService.getAllProducts(page);
            return Map.of("message", "查询成功", "data", products, "page", page);
        } catch (Exception e) {
            return Map.of("message", "查询错误：" + e.getMessage());
        }
    }
}
```

**规范要点**:
- 使用 `@RestController` 注解
- 使用 `@RequiredArgsConstructor` 实现依赖注入
- 基础路径使用 `/api` 前缀
- 返回类型统一使用 `Map<String, Object>` 或 `Map<String, String>`
- 响应包含 `message` 字段描述操作结果
- 使用 `try-catch` 包裹业务逻辑，统一异常处理
- 使用构造器注入 (`private final`)

#### 3.3.3 服务层 (Service)

**接口定义**:
```java
public interface OrderService {
    /**
     * 创建订单
     * @param order 订单信息
     */
    int createOrder(Order order);
    
    Order getOrderById(String orderId);
    List<Order> getAllOrders();
}
```

**实现类**:
```java
@Service
public class OrderServiceImpl implements OrderService {
    
    @Autowired
    private OrderMapper orderMapper;

    @Override
    public int createOrder(Order order) {
        System.out.println(new Date() + ": run createOrder");
        try {
            return orderMapper.insertOrder(order);
        } catch (Exception e) {
            e.printStackTrace();
            return -1;
        }
    }
}
```

**规范要点**:
- 接口中方法使用 JavaDoc 注释
- 实现类使用 `@Service` 注解
- 方法执行前打印日志，格式: `System.out.println(new Date() + ": run 方法名")`
- 返回 `int` 表示操作影响行数，-1 表示失败

#### 3.3.4 Mapper 接口

```java
@Mapper
public interface ProductMapper {

    @Select("SELECT * FROM products LIMIT 20 OFFSET #{offset}")
    List<Product> selectProductsByPage(@Param("offset") int offset);

    /**
     * 根据商品名称查询商品（模糊查询）
     */
    @Select("SELECT * FROM products WHERE name LIKE CONCAT('%', #{name}, '%')")
    List<Product> selectProductsByName(@Param("name") String name);

    @Insert("INSERT INTO products (id, name, price, tags, description, stock, created_at, updated_at) " +
            "VALUES (#{id}, #{name}, #{price}, #{tags}, #{description}, #{stock}, NOW(), NOW())")
    int insertProduct(Product product);

    @Update("UPDATE products SET stock = stock - #{quantity} WHERE id = #{productId} AND stock >= #{quantity}")
    int deductStock(@Param("productId") String productId, @Param("quantity") int quantity);

    @Delete("DELETE FROM products WHERE id = #{productId}")
    int deleteProduct(@Param("productId") String productId);
}
```

**规范要点**:
- 使用 `@Mapper` 注解
- 使用注解方式编写 SQL (`@Select`, `@Insert`, `@Update`, `@Delete`)
- 复杂 SQL 添加 JavaDoc 注释
- 参数使用 `@Param` 注解
- SQL 中表字段使用下划线命名 (`created_at`)
- 插入/更新使用 `NOW()` 函数设置时间

## 4. API 响应规范

### 4.1 成功响应

```java
// 查询成功
return Map.of("message", "查询成功", "data", products, "page", page);

// 操作成功
return Map.of("message", "创建订单成功", "orderId", orderId);
```

### 4.2 失败响应

```java
// 参数错误
return Map.of("message", "创建订单错误：商品信息为空（错误代码：O-001）");

// 业务逻辑错误
return Map.of("message", "创建订单错误：商品库存不足，当前库存：" + product.getStock() + "（错误代码：O-005）");

// 异常捕获
return Map.of("message", "创建订单错误：" + e.getMessage());
```

**错误代码格式**: `模块前缀-三位数字` (如 `O-001`, `Co-001`)

### 6.2 订单ID生成

```java
@Override
public String generateOrderId() {
    // 格式: 年月日(8位) + 序号(5位) + 随机字母(5位)
    // 示例: 2026040100001ABCDE
    String currentDate = LocalDate.now().format(DateTimeFormatter.ofPattern("yyyyMMdd"));
    String key = "order:seq:" + currentDate;
    Long sequence = redisTemplate.opsForValue().increment(key);
    // 设置24小时过期
    if (sequence != null && sequence == 1) {
        redisTemplate.expire(key, 24, TimeUnit.HOURS);
    }
    String seqStr = String.format("%05d", sequence);
    String randomChars = generateRandomLetters();
    return currentDate + seqStr + randomChars;
}
```

## 8. 日志规范

- 使用 `System.out.println()` 打印方法执行日志
- 格式: `new Date() + ": run 方法名"`
- 参数打印: `new Date() + ": run getAllProducts, page=" + page`

## 9. 异常处理

- Controller 层统一使用 `try-catch` 包裹
- 异常信息返回: `Map.of("message", "操作错误：" + e.getMessage())`
- Service 层捕获异常后打印堆栈: `e.printStackTrace()`

## 10. 启动类

```java
@SpringBootApplication
@MapperScan("com.gzasc.aishopping.mapper")

        SpringApplication.run(AiShoppingApplication.class, args);

    }

```

**规范要点**:
- 使用 `@MapperScan` 指定 Mapper 扫描路径
- 启动后打印服务地址
