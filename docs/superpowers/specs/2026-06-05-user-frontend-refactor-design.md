# AI Shopping 用户端前端重构设计文档

> 日期: 2026-06-05
> 项目: AI-Shopping frontier-user 前端重构
> 风格: 轻量现代风 · 简约科技感

> 审查状态: 已根据 frontend-design 技能完成审查

---

## 1. 设计目标

- **简约** — 去除冗余装饰，内容即界面
- **科技感** — 深色对比 + 蓝色点睛 + 精确排版
- **层次感** — 清晰的信息分区和视觉权重
- **一目了然** — 最重要的信息最先被看到
- **即时反馈** — 所有操作都有明确的交互响应

### 2.2 排版

字体家族:
- 标题: `'Space Grotesk', 'Noto Sans SC', 'PingFang SC', sans-serif`
- 正文: `'DM Sans', 'Noto Sans SC', 'PingFang SC', sans-serif`
- 数字/数据: `'Space Grotesk', 'JetBrains Mono', monospace` (价格、订单号、物流单号)

Space Grotesk 具有独特字符形态 + 科技感; DM Sans 高度可读, 作为正文清晰舒适。

| 层级 | 字号 | 字重 | 行高 | 场景 |

| H1 | 24px | 700 | 1.3 | 页面标题 |
| H2 | 20px | 700 | 1.3 | 区块标题 |
| H3 | 16px | 600 | 1.4 | 卡片标题 |
| Body | 14px | 400 | 1.6 | 正文 |

```

```

## 5. 页面场景

### 5.1 AI 聊天购物

核心交互流程:
1. 用户输入消息 → 发送按钮变蓝(有内容时)
2. 消息气泡以 `fadeSlideUp 0.4s` 出现
3. AI 回复前显示三点弹跳动画
4. AI 回复商品推荐 → 商品卡片以 `slideInUp` 交错入场(间隔 0.1s)
5. 卡片悬浮上移 4px + 蓝色顶线
6. 点击卡片 → 右侧 Drawer 滑出商品详情
7. 点击"立即购买" → 下单 Drawer (含地址选择)
8. 提交 → Toast 成功反馈

### 5.2 订单管理

核心交互流程:
1. 进入页面 → 加载骨架屏 (shimmer)
2. 列表项交错入场 (stagger 0.08s)
3. 顶部筛选胶囊切换
4. 点击订单 → 进入订单详情页 (slideInRight)
5. 订单详情含完整信息 + 进度条
6. 操作按钮根据状态动态渲染:
   - 待支付: [取消订单] [去支付]
   - 已发货: [查看物流] [确认收货]
   - 已送达: [查看物流] [申请退货]
7. 确认操作 → pop 弹窗 → 确认后状态更新动画

### 5.3 地址管理

核心交互流程:
1. 地址列表卡片展示
2. 操作: 编辑/设为默认/删除
3. 删除触发确认弹窗
4. 设为默认 → Toast "已设为默认地址"

## 6. 动画系统

| 动画名 | 时长 | 曲线 | 场景 |
|--------|------|------|------|
| 动画名 | 时长 | 曲线 | 场景 |
|--------|------|------|------|
| `fadeSlideUp` | 300ms | ease-out | 聊天消息、卡片入场 |
| `slideInRight` | 300ms | ease-out | Drawer 滑出、页面前进 |
| `slideInUp` | 300ms | ease-out | 列表项 staggered 入场 |
| `pop` | 250ms | ease-out | 弹窗弹出 |
| `slideDown` | 300ms | ease-out | Toast 出现 |
| `ripple` | 400ms | ease-out | 按钮点击涟漪 |
| `shimmer` | 1.5s | ease-in-out infinite | 骨架屏加载 |
| `bounce` | 1.4s | ease | AI 打字指示器 |
| `countdown` | 3s | linear | Toast 倒计时条 |

所有动画遵循原则:
- 时长 150-300ms (除 shimmer/bounce/countdown)
- 使用 ease-out 缓出曲线, 不使用弹性/弹跳曲线
- 仅使用 transform 和 opacity 属性, 不触发重排
- 连续动画仅用于加载指示器, 不做装饰
- 支持 `prefers-reduced-motion`

## 8. 交互反馈矩阵

| 操作 | 反馈类型 | 动画 | 时效 |
|------|---------|------|------|
| 点击按钮 | Ripple 涟漪 | `ripple` 400ms | 即时 |
| 提交请求 | 按钮 Loading + Spinner | 禁用态 | 请求中 |
| 请求成功 | Toast 成功 | `slideDown` 300ms | 3s 自动消失 |
| 请求失败 | Toast 错误 | `slideDown` 300ms | 3s 自动消失 |
| 确认取消/删除 | 底部动作面板 | `slideUp` 300ms | 用户操作 |
| 页面切换 | 滑入/滑出 | `slideInRight` / `slideOutLeft` | 300ms |
| 列表加载 | 骨架屏 | `shimmer` 1.5s | 数据到达 |
| 状态变更 | 标签颜色+图标过渡 | 300ms background transition | 即时 |
| AI 等待 | 三点 bounce | `bounce` 1.4s | 回复到达 |
| 卡片悬浮 | 上移 4px + 增强阴影 | 200ms transform | hover |
| 空状态 | 插画 + 文案 + CTA | `fadeIn` 300ms | 数据为空 |
| 错误状态 | 错误图标 + 重试  | `fadeIn` 300ms | 请求失败 |
