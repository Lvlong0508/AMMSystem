# AI-Shopping 前端代码规格说明

## 1. 技术栈

- **框架**: Vue 3 (Composition API)
- **构建工具**: Vite
- **路由**: Vue Router 4
- **HTTP客户端**: Axios
- **CSS框架**: Tailwind CSS
- **图标**: 原生 SVG 或 Emoji

**规范要点**:
1. **模板部分**:
   - 文件顶部添加路径注释 `<!-- src/components/... -->`
   - 使用双引号包裹属性值
   - 指令使用简写形式 (`@click`, `:disabled`, `v-if`)
   - 多行属性时，每个属性占一行，保持对齐

2. **Script 部分**:
   - 使用 `<script setup>` 语法
   - 组合式函数命名: `use+组件名+Logic`
   - 文本常量命名: `组件名_TEXT`，导入时别名为 `T`
   - Props 和 Emits 使用运行时声明
   - 从组合式函数解构所需方法

3. **Style 部分**:
   - 使用 `scoped` 属性
   - 统一使用 `@import './组件名.css'` 引入外部样式

### 3.3 组合式函数 (Composables) 规范

```javascript
// useProductCard.js
import { ref } from 'vue'

export function useProductCardLogic(props, emit) {
  const parseTags = (tags) => {
    if (!tags) return []
    if (Array.isArray(tags)) return tags
    return []
  }

  const handleOrderClick = () => {
    emit('showOrderDialog', props.product)
  }

  return {
    parseTags,
    handleOrderClick
  }
}
```

**规范要点**:
- 函数名使用大驼峰: `useProductCardLogic`
- 接收 `props` 和 `emit` 作为参数
- 返回对象包含所有需要暴露给模板的方法和状态
- 内部使用 Vue 3 响应式 API (`ref`, `reactive`, `computed` 等)

### 3.5 API 层规范

```javascript
// api/request.js
import axios from 'axios'
import Swal from 'sweetalert2'

export const request = axios.create({
    baseURL: 'http://localhost:8080',  // 网关统一入口
    timeout: 30000
})

// 请求拦截器：自动添加 Sa-Token
request.interceptors.request.use(
    config => {
        const token = localStorage.getItem('satoken')
        if (token) {
            config.headers['satoken'] = token
        }
        return config
    },
    error => Promise.reject(error)
)

// 响应拦截器：处理 401 未授权
request.interceptors.response.use(
    response => response.data,
    error => {
        if (error.response?.status === 401) {
            localStorage.removeItem('satoken')
            Swal.fire({
                title: '未登录',
                text: '请先登录',
                icon: 'warning',
                confirmButtonText: '去登录'
            }).then(() => {
                window.location.href = '/login'
            })
        }
        return Promise.reject(error)
    }
)
```

## 6. 工具函数规范

```javascript
// utils/formatter.js
/**
 * 从 AI 回复中提取商品 ID
 * 支持格式：【商品ID: 1001】、[商品ID: 1001]
 */
export function extractProductIds(text) {
    const regex = /[【\[]商品ID[：:]\s*([a-zA-Z0-9\-]+)[】\]]/g
    const ids = []
    let match
    while ((match = regex.exec(text)) !== null) {
        const id = match[1]
        if (id) ids.push(id)
    }
    return [...new Set(ids)]
}

/**
 * 格式化日期
 */
export function formatDate(date) {
    if (!date) return '-'
    const d = new Date(date)
    return d.toLocaleDateString('zh-CN')
}

/**
 * 格式化价格
 */
export function formatPrice(price) {
    return `¥${price.toFixed(2)}`
}
```

**规范要点**:
- 使用 JSDoc 风格注释
- 使用具名导出 (`export function`)
- 函数名单词首字母小写，驼峰命名
- 提供默认返回值处理边界情况

## 9. 项目特定约定

### 9.1 商家端 vs 用户端

| 项目 | 主题色 | 包名 | API前缀 |
|------|--------|------|---------|
| frontier-user | 蓝色 (#3b82f6) | AI-Mart 智能购物平台 | `/api/user/**` |
| frontier-seller | 橙色 (#f59e0b) | AI-Mart-商家服务平台 | `/api/seller/**` |

### 9.2 组件结构模式

每个组件目录包含 4 个文件:
1. `组件名.vue` - 模板文件
2. `组件名.css` - 样式文件
3. `Text.js` - 文本配置
4. `use组件名Logic.js` - 逻辑组合式函数

### 9.3 API 端点

- 网关基地址: `http://localhost:8080`（通过 `request.js` 统一配置）
- 统一从 `request.js` 导入 axios 实例
- 用户端接口前缀: `/api/user/{service}/**`
- 商家端接口前缀: `/api/seller/{service}/**`
- 认证接口: `/api/user/auth/**` 和 `/api/seller/auth/**`
- 自动携带 Sa-Token 请求头进行身份认证
- 跨域通过网关统一处理
