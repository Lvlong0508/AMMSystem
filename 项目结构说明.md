# AI-Shopping 项目结构说明

本文档记录 AI-Shopping 项目的整体架构和模块划分。

## 项目概览

```
AI-Shopping/
├── AI-Shopping-frontier/           # 前端项目（Vue 3）
│   ├── frontier-user/              # 用户端 - AI-Mart 智能购物平台
│   └── frontier-seller/            # 商家端 - AI-Mart 商家服务平台
├── AI-Shopping-backend_Eureka/     # 后端项目（Spring Cloud 微服务）
│   ├── eureka-server/              # 服务注册中心（端口 8761）
│   ├── product-service/            # 商品服务（端口 8081）
│   ├── product-api/                # 商品服务 API 接口
│   ├── order-service/              # 订单服务（端口 8082）
│   ├── order-api/                  # 订单服务 API 接口
│   ├── contact-service/            # 联系人服务（端口 8083）
│   ├── contact-api/                # 联系人服务 API 接口
│   ├── logistics-service/          # 物流服务（端口 8084）
│   ├── logistics-api/              # 物流服务 API 接口
│   └── chat-service/               # AI 聊天服务（端口 8085）
└── AI-Shopping-backend/             # [已弃置] 原单体应用（端口 8080）
```

## 前端模块

### frontier-user（用户端）

- **技术栈**: Vue 3 + Vite + Tailwind CSS
- **职责**: 面向消费者的购物平台
- **核心功能**:
  - AI 智能对话购物
  - 商品浏览与搜索
  - 订单管理
  - 联系人/收货地址管理
  - 物流跟踪

### frontier-seller（商家端）

- **技术栈**: Vue 3 + Vite + Tailwind CSS
- **职责**: 面向商家的管理平台
- **核心功能**:
  - 商品管理（上架、编辑、库存）
  - 订单处理（发货、状态更新）
  - 物流管理

## 后端模块

### 微服务架构

| 服务 | 端口 | 职责 | 依赖 |
|------|------|------|------|
| eureka-server | 8761 | 服务注册与发现中心 | - |
| product-service | 8081 | 商品管理（CRUD、库存） | Eureka |
| order-service | 8082 | 订单管理（下单、状态流转） | Eureka, Redis, product-api |
| contact-service | 8083 | 联系人/收货地址管理 | Eureka |
| logistics-service | 8084 | 物流跟踪、发货管理 | Eureka |
| chat-service | 8085 | AI 智能对话（通义千问） | Eureka, product-api |

### API 接口模块

| 模块 | 说明 | 用途 |
|------|------|------|
| product-api | 商品服务 Feign 接口定义 | 供 order-service、chat-service 调用 |
| order-api | 订单服务 Feign 接口定义 | 供其他服务调用 |
| contact-api | 联系人服务 Feign 接口定义 | 供其他服务调用 |
| logistics-api | 物流服务 Feign 接口定义 | 供其他服务调用 |

### 服务间调用关系

```
order-service ──→ product-api ──→ product-service
chat-service ───→ product-api ──→ product-service
```

## 已弃置模块

### AI-Shopping-backend

- **状态**: 已弃置
- **说明**: 原单体应用版本，使用 Spring Boot 单体架构
- **替代**: AI-Shopping-backend_Eureka（微服务版本）
- **保留原因**: 历史版本参考
